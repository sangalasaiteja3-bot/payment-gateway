## 🎯 Problem Statement
In modern digital systems, payment gateways are used to process and manage transactions securely and efficiently.  
This project develops a **menu-driven payment gateway system** using C and data structures to handle transaction records with CRUD operations.

---

## 🎯 Objectives
- Implement data structures in a real-world scenario  
- Perform CRUD operations efficiently  
- Use dynamic memory allocation  
- Develop modular programming using functions  

---

## 🌍 Real-world Application
- Online payment systems (UPI, Cards)  
- E-commerce platforms  
- Ticket booking systems  
- Banking applications  

---

## 🧠 Data Structure Used
**Singly Linked List**

Each node contains:
- Transaction ID  
- Amount  
- Payment Method  
- Status  
- Pointer to next node  

---

## ⚙️ Features (CRUD Operations)

| Operation | Description |
|----------|------------|
| Create   | Add new transaction |
| Read     | Display all records |
| Update   | Modify transaction |
| Delete   | Remove transaction |
| Search   | Find by ID |

---

## 🔄 Algorithm
1. Start program  
2. Display menu  
3. Take user input  
4. Perform selected operation  
5. Repeat until exit  

---

## 🛠️ Technology Used
- C Programming  
- Structures (`struct`)  
- Dynamic Memory (`malloc`, `free`)
